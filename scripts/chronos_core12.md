## 6. 下一季度 4 个观察重点

1. **HBM4 Q2/Q3 财报兑现**：MU / SNDK——"产能售罄"故事能否持续，一旦兑现节奏放慢，估值消化会很快。
2. **1.6T 光模块渗透率拐点**：LITE / COHR / CIEN——当前股价已经把"量"和"毛利"打进预期，**下季度要看订单持续性**。
3. **核能夏季用电高峰 + 表后 PPA 新单**：VST 的拐点信号。任何一笔新的核电 PPA 大单都能让 VST 重拾动能。
4. **Neocloud 分化**：NBIS 能否从欧洲独立叙事拿到独立定价；如果 CRWV 出现 IPO / 新融资动作，NBIS 是最直接的受益标的。

## 8. 已放弃的板块 / 标的（刻意不跟）

- **Battery & Storage 整块**（EOSE / FLNC / SLDP）：Q1 −37.83%，逻辑被证伪，不是 AI 基础设施的硬约束。
- **Compute Mining 大部分**（IREN / HUT / CIFR）：电力额度套利故事短，不适合做中期跟踪。
- **Manufacturing 整块**（CLS / FLEX / FN / SANM）：跟头部客户（Nvidia / DELL）高度同步，看上游即可。
- **Hyperscale Cloud 里的 MSFT / GOOGL / ORCL**：这三只不是"专门的 AI infra 标的"，日常本来就在看；不纳入 AI Infra 跟踪池。
- **Energy Infra 里的 CEG / OKLO / LEU / TLN / GEV / BE**：核能主题留 VST 一个种子仓就够；BE / GEV 放观察区。
- **Chip Design 里的 ARM / ASML / AMD / INTC**：ARM 估值打满、ASML 低频新闻、AMD 永远第二名、INTC 政策驱动难跟踪。
- **Optics 里的 AAOI / SMTC / ALAB**：AAOI 属于困境反转投机；SMTC 生态弱；ALAB 进观察。
- **Memory 里的 PSTG / STX**：PSTG 故事和 SNDK 重叠；STX 进观察。
- **Neocloud 里的 APLD / GLXY / NUAI / CRWV**：APLD 偏地产式；GLXY / NUAI 风险篮子；CRWV 和 NVDA 绑定过深。

**放弃的理由归一句话**：**要么故事被证伪、要么故事对但没跟踪价值、要么在 Core-12 的代表身上已覆盖**。

## 9. 技术细节
